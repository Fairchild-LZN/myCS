# 101 对称二叉树

## 不爱睡觉的大猪

```C++
/*
镜像对称：递归定义

左孩子 == 右孩子
左孩子的左子树 == 右孩子的右子树
左孩子的右子树 == 右孩子的左子树
*/
class Solution{

    bool dfs(TreeNode* a, TreeNode* b){
        if(!a && !b) return true;
        if(!a || !b || a->val != b-> val) return false;
        return dfs(a->left, b->right) && dfs(a->right, b->left);
    }
    
    bool isSymmetric(TreeNode* root){
        if(!root) return true;

        return dfs(root->left, root->right);
    }
};
```
