# 538 将二叉树转换成累加树

## 不爱睡觉的大猪

```C++
class Solution{
    public:
    int sum = 0; // 前缀和
    TreeNode* convertBST(TreeNode* root){
        if(!root) return root;

        convertBST(root -> right);
        sum += root -> val;
        root -> val = sum;
        convertBST(root -> left);
        return root;
    }
};
```
