# 338 比特位计数

## 不爱睡觉的大猪

```C++
class Solution{
    public:
    vector<int> countBits(int n){
        vector<int> ret(n+1);
        ret[0] = 0;
        for(int i = 1; i <= n; i++){
            if(i & 1){
                ret[i] = ret[i-1] + 1;
            }else{
                ret[i] = ret[i >> 1];
            }
        }
        return ret;
    }
};
```
