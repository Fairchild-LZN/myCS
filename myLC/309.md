# 309 最佳股票买卖时机含冷冻机

## 不爱睡觉的大猪

```C++
/*
f[i][0] 第i天不持有股票的最大收益
f[i][1] 第i天持有股票的最大受益

答案：f[n][0]

不持有股票：前一天也不会持有：当天卖掉
f[i][0] = max(f[i-1][0], f[i-1][1] + p[i-1])

持有股票：前一天已经持有：当天买入（要求前两天是没有股票的）
f[i][1] = max(f[i-1][1], f[i-2][0] - p[i-1])
*/

class Solution{
    public:
    int maxProfit(vector<int>& p){
        int n = p.size();
        vector<vector<int>> f(n+1, vector<int>(2, 0));

        f[1][0] = 0;
        f[1][1] = -p[0];

        for(int i = 2; i <= n; i++){
            f[i][0] = max(f[i-1][0], f[i-1][1] + p[i-1]);
            f[i][1] = max(f[i-1][1], f[i-2][0] - p[i-1]);
        }

        return f[n][0];
    }
};
```
