# 021 合并两个排序的链表

## 不爱睡觉的大猪

1. 不允许修改链表结构，则新建链表节点（较易）
2. 允许修改链表结构，不能新建链表节点（较难）

```C++
class Solution{
    public:
    ListNode* meregeTwoLists(ListNode* l1, ListNode* l2){
        if (!l1 && !l2) return nullptr;
        else if (!l1) return l2;
        else if (!l2) return l1;

        ListNode* head = nullptr; // 合并后的链表头节点
        ListNode* tail = nullptr; // 合并过程中，结果链表的尾节点

        while(l1 && l2){
            if (l1 -> val < l2 -> val){
                // 将当前 l1的节点追加到结果链表的尾部
                if (tail){
                    tail -> next = l1;
                }
                tail = l1;
                if (!head){
                    head = l1;
                }
                l1 = l1 -> next;
            }else {
                if (tail){
                    tail -> next = l2;
                }
                tail = l2;
                if (!head){
                    head = l2;
                }
                l2 = l2 -> next;
            }
        }

        // l1 l2 其中一方走完，另一方没走完
        if(l1){
            tail -> next = l1;
        }
        if(l2){
            tail -> next = l2;
        }

        return head;
    }
}
```
