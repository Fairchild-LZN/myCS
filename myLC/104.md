# 104 二叉树的最大深度

## 不爱睡觉的大猪

```C++
class Solution{
    int ret;

    void dfs(TReeNode* root, int depth){
        if(!root) return;
        ret = max(ret, depth);
        dfs(root->left, depth+1);
        dfs(root->right, depth+1);
    }

    int maxDepth(TreeNode* root){
        ret = 0;
        dfs(root, 1);
        return ret;
    }
};
```
