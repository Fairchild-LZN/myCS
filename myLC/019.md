# 019 删除链表的倒数第N个结点

- 注意被删除的是头节点

## 不爱睡觉的大猪

1. 计算第Len - N + 1的位置就是要被删除的
2. 准备两个指针LR，R指针先移动N次，然后同时移动LR指针，当R指针到达链表底后，L指针所指的就是要删除的

```C++
class Solution{
    public:
    ListNode* removeNthFormEnd(ListNode* head, int n){
        if (!head) return nullptr;
        ListNode* HEAD = new ListNode();
        HEAD -> next = head;

        ListNode* L = head;
        ListNode* R = head;

        for (int i = 0; i < n; i++){
            R = R -> next;
        }

        ListNode* pre = HEAD;
        while(R){
            R = R -> next;
            L = L -> next;
            pre = pre -> next;
        }
        pre -> next = L -> next;
        return HEAD -> next;
    }
};
```
