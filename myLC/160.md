# 160 相交链表

## 不爱睡觉的大猪

```C++
class Solution{
    public:
    ListNode *getIntersectionNode(ListNode *a, ListNode *b){
        ListNode *pa = a, *pb = b;

        // 如果不相交， pa=pb=nullptr
        // 如果相交，pa=pb=T
        while(pa != pb){
            if(!pa) pa = b;
            else pa = pa -> next;
            if(!pb) pb = a;
            else pb = pb -> next;
        }
        
        return pa;
    }
};
```
